# quickcart-sales-analysis
End-to-End Data Analyst Project using Python, MySQL, and Power BI

## Project Overview
This project analyzes QuickCart e-commerce order data to identify sales trends, delivery performance, inventory risks, and payment behavior.

## Tools Used
Python (Pandas)
MySQL
Power BI

## Project Workflow
1. Data Cleaning using Python
2. Data stored in MySQL
3. Business Analysis using SQL queries
4. Data Visualization using Power BI dashboard

## Business Questions
- Which product categories generate the highest revenue?
- Which cities generate the most orders?
- Which delivery partner performs fastest?
- Are stock shortages affecting sales?
- Which payment methods are most used?

## Key Insights
- Total Revenue: 503,233
- Total Orders: 1200
- Top Category: Bakery
- Top City: Delhi
